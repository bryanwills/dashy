##Dashy - A Personal Python Project

####Dashy is being created as a way to manage servers both on-prem and cloud servers. I wanted to create a dashboard of some sort that will allow me to have an overall view of what status my servers are in and the ability to start a shell for any of the servers listed. I have a number of servers locally and cloud based using various email accounts. Having a central location for all of them makes it easier for me to manage.
